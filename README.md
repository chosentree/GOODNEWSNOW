# /今日与恩典同行/

### 第172 06/21 礼拜二

>从他丰满的恩典里，我们都领受了，而且恩上加恩。律法本是借着摩西传的，恩典和真理都是由耶稣基督来的。——约翰福音1:16-17

你如何看待恩典和真理的关系？有的人全讲神的恩典却忽视了圣经的真理。但圣经告诉我们，恩典和真理不可分割。耶稣说：“你们若爱我，就必遵守我的命令。

所以，在恩典的福音之下并不表示顺服神的真理可有可无。神不再以好行为作蒙他悦纳的条件，并不表示他不再要求公义的生活，因为圣经教导的行事原则既荣耀神，同时又保护他百姓免受伤害。

神告诫我们不要把他无条件的爱和恩典当作自我放纵的借口。因为这种自我中心的生活最终会让我们背负罪孽和罪的后果，而这本是神要借他的恩典除去。

**祷告：天父，愿我今日服侍你之时，耶稣基督的恩典和真理同时在我生命彰显。**
